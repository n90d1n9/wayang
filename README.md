# Gollek Extension for Agentic AI

This is the extension for Gollek Agentic AI. It is a collection of standalone modules that can be used to extend the functionality of Gollek Agentic AI and depend on Gollek SPI modules.

## Modules

- agent-core: Core agent logic
    - This module contains the core agent logic.
- a2a: Agent-to-agent communication
    - This module contains the agent-to-agent communication logic.
- mcp: MCP server
    - This module contains the MCP server logic.
- memory-pgvector: Memory storage
    - This module contains the memory storage logic.
- agent-skills: Agent skills
    - This module contains the agent skills logic.
- agent-skills-command: Agent skills command
    - This module contains the agent skills command logic.
